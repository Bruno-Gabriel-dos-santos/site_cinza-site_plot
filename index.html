Transformar foto rgb em cinza

A transformação de uma imagem rgb ( colorida ) para uma imagem monocromatrica ( preto e branco ) ocorre quando os pixeis no formato rgb são convertidos em uma unidade de escala de luminosidade .
Para criar essa escala de luminosidade o pardrao (r,g,b) tem suas variações de intensidade modificados por uma formula que calcula os pesos individuais de cada cor rgb e os soma a fim de criar um pixel unico monocromatico.
O site em questão usa um modelo padrão dessa formula , porem é possivel customiza-la a fim de evidenciar ou focar detalhes diferentes na foto.

Para Transformar a sua imagem colorida em uma imagem monocromatica basta arrastar a sua imagem escolhida para o campo abaixo ou selecionar a imagem clicando no campo indicado.
Apos selecionada a imagem basta apenas clicar no botão "Aplicar efeito" abaixo do campo fotos.
